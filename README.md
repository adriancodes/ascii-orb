# ascii-orb

Animated ASCII orb component for React — 16 variants, ripples, fully customizable.

> Work in progress: full documentation, live showcase, and npm release are on the way.

```tsx
import { AsciiOrb } from "ascii-orb";

<AsciiOrb variant="aether" />;
```

Headless engine (no React) available at `ascii-orb/core`.
